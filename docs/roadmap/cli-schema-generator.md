# CLI Schema Generator

**Status:** Implemented
**Priority:** Medium
**Package:** `@testurio/cli`

## Overview

Generate type-safe TypeScript schemas from OpenAPI specifications for use with Testurio's `HttpProtocol`. Uses Orval under the hood for Zod schema generation, then builds a Testurio-compatible operations map.

## Goals

- Single command: `testurio generate` generates everything
- Type-safe output with Zod schemas and operations map
- Minimal abstraction - thin wrapper around Orval
- Extensible for future gRPC/GraphQL support

## Architecture

```
CLI Layer (commander.js)
  │  - Entry point, argument parsing, console output
  │  - Commands: generate, init
  ▼
Generation Layer
  - Config loading (cosmiconfig + Zod validation)
  - Orval invocation for Zod schema generation
  - Operations map generation (pure functions)
```

## CLI Usage

```bash
# Generate with default config (testurio.config.ts)
testurio generate

# Generate with custom config
testurio generate -c ./custom.config.ts

# Generate with inline options
testurio generate -i ./api.yaml -o ./generated.ts

# Initialize config file
testurio init
```

## Configuration

```typescript
// testurio.config.ts
import { defineConfig } from '@testurio/cli';

export default defineConfig({
  input: './api/openapi.yaml',
  output: './src/generated/api.ts',
  zod: {
    strict: { response: true, body: true },
    coerce: { query: true, params: true },
  },
  operationsMap: {
    typeHelpers: true,
    errorSchemaName: 'apiErrorSchema',
  },
});
```

## Generated Output

The generator produces a single TypeScript file containing:

1. **Zod schemas** - Generated by Orval from the OpenAPI spec
2. **Operations map** - Maps operation IDs to request/response definitions
3. **Type helpers** (optional) - Utility types for extracting request/response types

```typescript
// Generated output (simplified)
export const operations = {
  getUsers: {
    request: { method: 'GET' as const, path: '/users' as const, query: getUsersQuerySchema, body: zod.never() },
    response: { 200: zod.array(userSchema), 500: apiErrorSchema },
  },
  createUser: {
    request: { method: 'POST' as const, path: '/users' as const, query: zod.never(), body: createUserBodySchema },
    response: { 201: userSchema, 400: apiErrorSchema },
  },
} as const;

export type ApiOperations = typeof operations;
export type OperationId = keyof ApiOperations;
```
